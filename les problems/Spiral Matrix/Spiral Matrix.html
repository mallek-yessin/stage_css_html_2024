Spiral Matrix

Ce code Python définit une fonction nommée "spiralOrder" qui prend une matrice en entrée et retourne ses éléments dans l'ordre spiralé, c'est-à-dire en suivant un motif de spirale. Il utilise un processus itératif pour parcourir la matrice, en ajustant les limites à chaque étape pour couvrir les éléments restants dans la spirale. À chaque itération, il parcourt une rangée ou une colonne dans un sens spécifique, ajoutant les éléments rencontrés à une liste résultante. Une fois que tous les éléments de la matrice ont été parcourus, la liste contenant les éléments dans l'ordre spiralé est retournée. Cette approche permet de résoudre efficacement le problème de parcours en spirale d'une matrice en utilisant un algorithme itératif simple. 


    def spiralOrder(self, matrix):
        """
        :type matrix: List[List[int]]
        :rtype: List[int]
        """
        k=0
        w=1
        l=[]
        p=len(matrix)
        h=len(matrix[0])
        i=0
        while len(l)<(len(matrix[0])*len(matrix)):
            for j in range (k,h):
                            l.append (matrix[i][j])

            
            if len(l)<(len(matrix[0])*len(matrix)):
                for i in range (w,p):
                            l.append (matrix[i][j])

            p=p-1
            h=h-1
            j=h-1
            if len(l)<(len(matrix[0])*len(matrix)):
                while j>k-1:
                            l.append (matrix[i][j])
                            j=j-1

            j=j+1
            i=p-1
            if len(l)<(len(matrix[0])*len(matrix)):
                while i>w-1:
                            l.append (matrix[i][j])
                            i=i-1
            i=i+1
            k=k+1
            w=w+1
            
        return l
