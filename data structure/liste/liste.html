liste

Il y a des fonctions près définie sont:

append(), extend(), insert(), remove(), pop(), sort(), reverse()

a)append()

qui ajoute un element a la fin d’une liste

        
            my_list = [1, 2, 3]
            my_list.append(4)
            print(my_list)  # [1, 2, 3, 4]

b)extend()

ajoute une autre liste a la fin d’une liste

           
            my_list = [1, 2, 3]
            my_list.extend([4, 5, 6])
            print(my_list)  # [1, 2, 3, 4, 5, 6]

c) insert()

Insère un élément à une position spécifiée

              
                my_list = [1, 2, 3]
                my_list.insert(1, 'a')
                print(my_list)  # [1, 'a', 2, 3]

d) remove()

Supprime la première occurrence d'un élément spécifié.

                 
                    my_list = [1, 2, 3, 2, 4]
                    my_list.remove(2)
                    print(my_list)  # [1, 3, 2, 4]

e) pop()

Supprime et retourne l'élément à une position spécifiée (par défaut, le dernier élément).

                    
                        my_list = [1, 2, 3]
                        popped_element = my_list.pop()
                        print(popped_element)  # 3
                        print(my_list)         # [1, 2]
                        
                        popped_element = my_list.pop(0)
                        print(popped_element)  # 1
                        print(my_list)         # [2]

f)reverse()

Inverse l'ordre des éléments de la liste.

                       
                        my_list = [1, 2, 3]
                        my_list.reverse()
                        print(my_list)  # [3, 2, 1]
