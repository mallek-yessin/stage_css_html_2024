dictionner

Il y a des fonctions près définie sont:

keys(), values(), items(), get(), update(), pop(), popitem()

a) keys() :

Retourne une vue sur les clés du dictionnaire

        
            my_dict = {'a': 1, 'b': 2, 'c': 3}
            keys = my_dict.keys()
            print(keys)  # dict_keys(['a', 'b', 'c'])

b) values()

Retourne une vue sur les valeurs du dictionnaire.

           
            my_dict = {'a': 1, 'b': 2, 'c': 3}
            values = my_dict.values()
            print(values)  # dict_values([1, 2, 3])

c) items()

Retourne une vue sur les paires clé-valeur du dictionnaire.

              
                my_dict = {'a': 1, 'b': 2, 'c': 3}
                items = my_dict.items()
                print(items)  # dict_items([('a', 1), ('b', 2), ('c', 3)])

d) get(key[, default])

Retourne la valeur associée à la clé spécifiée, ou la valeur par défaut si la clé n'existe pas.

                 
                    my_dict = {'a': 1, 'b': 2, 'c': 3}
                    value = my_dict.get('b', 0)  # 2

e) update()

Met à jour le dictionnaire avec les paires clé-valeur d'un autre dictionnaire ou d'une séquence de paires clé-valeur.

                    
                        my_dict = {'a': 1, 'b': 2}
                        my_dict.update({'b': 3, 'c': 4})
                        print(my_dict)  # {'a': 1, 'b': 3, 'c': 4}

f)pop(key[, default])

Supprime et retourne la valeur associée à la clé spécifiée, ou la valeur par défaut si la clé n'existe pas.

                       
                        my_dict = {'a': 1, 'b': 2, 'c': 3}
                        value = my_dict.pop('b')  # 2

g) popitem()

Supprime et retourne une paire clé-valeur arbitraire du dictionnaire.

                          
                            my_dict = {'a': 1, 'b': 2, 'c': 3}
                            key, value = my_dict.popitem()
