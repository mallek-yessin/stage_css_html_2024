tuple

il y a les fonctions sorted , min , max et sum dans le tuple

any()

Retourne True si au moins un élément du tuple est vrai (ou est évalué comme vrai). 

    
        my_tuple = (0, 0, 1)
print(any(my_tuple))  # True

all()

Retourne True si tous les éléments du tuple sont vrais (ou sont évalués comme vrais) 

    
        my_tuple = (1, 2, 3)
        print(all(my_tuple))  # True

count(x)

Retourne le nombre de fois que l'élément x apparaît dans le tuple. 

    
        my_tuple = (1, 2, 2, 3)
        print(my_tuple.count(2))  # 2

index(x)

Retourne l'index de la première occurrence de l'élément x dans le tuple. 

    
        my_tuple = (1, 2, 3)
        print(my_tuple.index(2))  # 1
